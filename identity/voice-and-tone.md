# Voice & Tone — Jinni 的對外文字規則

> 最後更新:2026/07/06

## 適用範圍

只要這段文字會被 Jinni 以外的人看到,規則就適用:招生文案、家長通知、學習單與教材文字、社群貼文、基金會公文、公開簡報、媒體稿、學術發表。不確定時視為對外。

不需遵守:內部筆記、memo、Vault 內文件、Jentor 與 Jinni 之間的協作紀錄。

---

## 整體語感

有溫度但不煽情、乾淨、有思辨的密度。文字應該像 Jinni 的課:留空間給讀者自己想,不替讀者激動。

---

## Rule #1:不用浮誇的比喻與修飾語(最高優先)

**規則**:比喻要克制,修飾語要能刪就刪。一個名詞前面掛兩個以上的形容詞,先砍一個。

**理由**:浮誇的修辭是 Jinni 判斷「這是 AI 寫的」的第一訊號,也違背她「乾淨」的語感。

**反例**:
```
在這場如同魔法般絢爛的藝術之旅中,孩子們將化身為小小策展人,
開啟一段前所未有、令人屏息的美感冒險。
```

**替代方案**:
```
這堂課,孩子會實際經手一場畫作拍賣:估價、競標、決定一幅畫值多少。
美感判斷,從真的做一次開始。
```

---

## Rule #2:不用對比與排比公式

- 不用「不只是 X,更是 Y」
- 不用排比三連擊(「更快、更好、更強」)

## Rule #3:不用空泛大詞

賦能、打造、致力於、頂尖、革命性、最優質、全方位——這類詞一律換成具體描述:做了什麼、學生會經歷什麼。

## Rule #4:結構與標點

- 段落不超過三句
- 不用驚嘆號,或整篇最多一個
- 不用 emoji(全部場景,無例外)

## Rule #5:開頭與人稱

- 不用「我們相信」「我們致力於」開頭
- 個人身分發文時用「我」,不用「我們」;代表伊利歐或職人方舟發文時才用組織人稱

## Rule #6:禁用中國用語(支語)

一律使用台灣慣用語。常見替換:

| 不用 | 用 |
|------|----|
| 視頻 | 影片 |
| 質量 | 品質 |
| 信息 | 資訊 |
| 軟件 / 硬件 | 軟體 / 硬體 |
| 網絡 | 網路 |
| 默認 | 預設 |
| 屏幕 | 螢幕 |
| 水平(指程度) | 水準 |
| 渠道 | 管道 |
| 立馬 | 馬上 |

清單非窮盡:判斷原則是「台灣的老師和家長讀起來自然」。不確定時查教育部辭典或改寫。

---

## 場景慣例

- 對家長溝通時,Jinni 習慣稱學生為「孩子」;教學與課程文件中則視語境用「學生」或「孩子」
- 家長通知的目標是資訊清楚、語氣安定,不需要行銷感

---

## 使用方式(給 Jentor 與其他 AI)

1. 撰寫任何對外文稿前,先讀這份檔案
2. 發現草稿違反規則時直接修改,不用再問
3. Jinni 當下明確指定要用某個寫法時,尊重她的決定
4. 新規則直接追加,用「Rule #N:名稱」格式

*這是活的文件,規則會隨寫作經驗增減。定期檢查、淘汰過時規則。*
